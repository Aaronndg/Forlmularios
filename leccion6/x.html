Lección 6: Listas desplegables (<select>, <option>)

Definición y Propósito

Las listas desplegables en HTML se crean utilizando las etiquetas <select> y <option>. Estas permiten a los usuarios seleccionar una opción de un conjunto predefinido de opciones, lo que es útil para mantener la interfaz limpia y organizada.

Uso Común

Las listas desplegables son ideales para:

  • Seleccionar un país o región.
  • Elegir una categoría de producto.
  • Seleccionar un rango de precios.
  • Elegir una fecha o un mes específico.

Ejemplo Básico

A continuación, se muestra un ejemplo básico de una lista desplegable:

<select name="fruits">
    <option value="apple">Manzana</option>
    <option value="banana">Banana</option>
    <option value="orange">Naranja</option> 
    <option value="grape">Uva</option>
</select>
